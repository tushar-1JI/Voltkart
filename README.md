# ⚡ Voltkart By "TUSHAR SHARMA"

Voltkart is a fast, modern grocery delivery web app built for a seamless instant-delivery experience — think Blinkit, but with its own identity.

## What it does
Voltkart lets users browse products, add them to cart, place orders, and track deliveries in real time — all through a clean and responsive interface. There's also a full admin panel to manage orders and assign riders on the go.

## Features
- 🛒 Smart product listing with cart management
- 🔐 Secure login and user profile system
- 📦 Real-time order tracking
- 🗺️ Live map integration using Leaflet.js
- 🛵 Dedicated rider and admin panel
- 💾 Persistent data with LocalStorage

## Tech Stack
- HTML, CSS, JavaScript
- Leaflet.js for live maps
- LocalStorage for client-side data
